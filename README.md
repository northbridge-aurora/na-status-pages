# Northbridge Aurora Status Pages

Configuration and templates for the public-facing status pages of Northbridge Aurora services, including the **EggTrail Platform**.

## Services Monitored
- `api.northbridge-aurora.xyz` (Core API)
- `edge.northbridge-aurora.xyz` (Edge Agents)
- `cdn.northbridge-aurora.xyz` (Static Assets & Media Kits)

## Incident History
Past incident reports are archived in the `history/` directory. For current status, visit [status.northbridge-aurora.xyz](https://status.northbridge-aurora.xyz).
